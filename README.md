Title: Github Repo reader

Summary:

Repository url: https://github.com/abusaeed2433/JavaInREADME

Use this repository as the data source. Then create a spring boot application having some get endpoints only. The endpoints are as follows:

    1. /api/v1/read_contributions: Return the contributors of the repository as response.

    2. /api/v1/read_indices: Return the indices. See example response.

    3. /api/v1/read_blog?topic_name={x}&sub_topic_name={y}: Return the blog info.


You can see the example response using this base url: https://apiendpoint.site/jir/
